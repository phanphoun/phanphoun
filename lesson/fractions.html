មេរៀន៖ ភាគ (Fractions)

ទំព័រដើម

អត្ថបទមេរៀន

ភាគ គឺជាចំនួនដែលសរសេរជា លេខខាងលើ (ញែក) និង លេខខាងក្រោម (បែង) ដោយមានខ្សែបន្ទាត់កណ្ដាល។ ឧទាហរណ៍៖ ១/២, ៣/៤, ៥/៨។

ប្រែភាគ ➜ ភាគរយ

  • ១/២ = ០.៥ = ៥០%
  • ១/៤ = ០.២៥ = ២៥%
  • ៣/៤ = ០.៧៥ = ៧៥%
គន្លឹះ៖ បែង (denominator) ៤ ឬ ២ អាចបម្លែងទៅទសភាគងាយស្រួល, ប្រែទៅ % ដោយគុណ ១០០%.

បូកភាគ (បែងដូចគ្នា)

បូក ១/៨ + ៣/៨ = (១ + ៣) / ៨ = ៤/៨ = ១/២

ចម្លើយ៖ ១/២

លំហាត់

  • កាត់អំពើ ៤/៨ ទៅទម្រង់សាមញ្ញ
  • បម្លែង ២/៥ ទៅភាគរយ
  • បូក ១/៤ + ១/២
  1. ៤/៨ = ១/២
  2. ២/៥ = ០.៤ = ៤០%
  3. ១/៤ + ១/២ = ១/៤ + ២/៤ = ៣/៤
