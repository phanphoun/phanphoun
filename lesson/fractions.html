មេរៀន៖ ភាគ (Fractions)

សូមស្វាគមន៍មកកាន់មេរៀនភាគ

ស្វែងយល់អំពីភាគ

រៀនអំពីភាគ ការប្រែប្រួល និងប្រតិបត្តិការជាមួយភាគជាមួយនឹងឧទាហរណ៍ងាយស្រួល

អត្ថបទមេរៀន

📈

ឧទាហរណ៍ការប្តូរ

  • ១/២ = ០.៥ = ៥០%
  • ១/៤ = ០.២៥ = ២៥%
  • ៣/៤ = ០.៧៥ = ៧៥%
គន្លឹះ៖ បែង (denominator) ៤ ឬ ២ អាចបម្លែងទៅទសភាគងាយស្រួល, ប្រែទៅ % ដោយគុណ ១០០%.

➕ បូកភាគ (បែងដូចគ្នា)

រៀនពីការបូកភាគដែលមានបែងដូចគ្នា

🧮

ឧទាហរណ៍ការបូក

បូក ១/៨ + ៣/៨ = (១ + ៣) / ៨ = ៤/៨ = ១/២

ចម្លើយ៖ ១/២

📝 លំហាត់

សាកល្បងចំណេះដឹងរបស់អ្នក

សំណួរ

  • កាត់អំពើ ៤/៨ ទៅទម្រង់សាមញ្ញ
  • បម្លែង ២/៥ ទៅភាគរយ
  • បូក ១/៤ + ១/២
  1. ៤/៨ = ១/២
  2. ២/៥ = ០.៤ = ៤០%
  3. ១/៤ + ១/២ = ១/៤ + ២/៤ = ៣/៤
